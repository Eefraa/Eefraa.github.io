（一）

电梯门打开合上。

她手指要按。看见亮起的楼层收了回去。樊振东看见扣在她手指上的戒指，亮闪得让她手指看起来不堪负重。

她穿着帆布凉鞋，小巧的脚趾染着黑色。樊振东瞧见左脚小腿的淤青化为即将展翼的鸟。

“你们就是新户？”她笑眯眯转头跟电梯角依偎着的俩个人。男人似乎才开完会，还穿着白色的衬衫和西裤。

他身材很好，肌肉结实，挽起的衬衣袖下是有力的小臂。虽然皮肤白皙，但半点不羸弱。孙颖莎颇为满意的点头。

双眼皮大眼睛鹅蛋脸的女人面容姣好。穿着吊带碎花裙，黑色顺滑的长发落到腰间。眉尖轻轻一蹙，我见犹怜。她有些防御地往他身后藏。并不理会孙颖莎的搭话。

樊振东眯起眼睛，与她目光交接。黑色的背心下面是一条简单的亚麻短裤，白嫩嫩俩条腿笔直又匀称。

“是的。你好。”他哪知道她葫芦里藏的什么药。不过退役几年，连人都不认识了吗？

孙颖莎笑着说你好。到肩膀一刀切的黑发随着她的动作轻轻晃悠。

樊振东看见她背心露出的胸前，大片白花花的肌肤裸露着。幽深的沟壑若隐若现，她还是坠着从前的玉佛，愈发通透的绿色。

孙颖莎看着他滚动的喉结，意味不明笑了。

一梯俩户。孙颖莎看着他俩先走出电梯，然后在她目光里打开旁边的门。

回到家，把鞋子踢掉。光脚踩到吧台打开冰箱，拿出冷藏好的西瓜，坐在沙发边吃边看电视。

突然想起。

电梯里那个瘦了，五官挺立，目光冷漠又带着打量的人不怪那么眼熟。

门口传来指纹解锁的声音，孙颖莎跳下沙发往玄关跑。门一打开，她树袋熊一样四肢缠着门口的人。

傅清风抱住她，亲亲她颈窝：“又吃冰的，又不穿鞋。”

孙颖莎胡乱在他脸颊边亲亲，刚想回话就看见拎着一袋垃圾出来的樊振东，站在门口与她四目相接。

“没有。”他笑着摸摸她柔顺的黑色长发，“饿了吧？我给你做饭。”

傅清风笑着就她的勺子挖西瓜吃：“自己队友都没能第一时间认出来吗？”

“这是我和我太太的心意。初次见面，多包涵。”樊振东把东西递过去。

孙颖莎点头，抢在傅清风前面接过餐盘。她指尖冰凉，软软似无端擦过他温热的手指。樊振东眉心一跳，看她歪头笑。

“客气啦！要不今天来我家吃饭好啦！”她热情又大方，活像做饭是她拿手好戏。可是樊振东明明知道，她十指不沾阳春水。

傅清风看她的眼神无奈又宠溺，樊振东牵过蓝旻的手。“我和我太太要去看歌剧。今天不巧了。”

孙颖莎突然笑出声。一副了然的点头。傅清风捏捏她后颈：“那下次约。”

蓝旻抿着唇被他牵着往家里走。心跳得有些不正常。樊振东看她脸色有些发白，俯下身把头发勾在她耳后，“怎么了？”

她焦虑的抱他满怀，“我很爱你，樊振东。”

他笑着轻轻拍她背：“好。”

孙颖莎拍拍手心的曲奇碎，撅着嘴不再吃了。傅清风尝了俩片，觉得味道不错。

“你邀请人家吃饭，也不准备吗？”

“到时候让人送餐过来不就好了吗？”孙颖莎想到他们要去看歌剧，趴在沙发上笑得见牙不见眼：“可是他们好高级趣味。不像我，跟你看了那么多次歌剧，都一点兴趣没有。”

傅清风拿曲奇的手顿了一下：“我现在也不喜欢。”

（二）

孙颖莎闭上眼睛，突然想起樊振东笔挺的黑色西裤。樊局成真了？她拿起手机，推送樊振东之前的比赛视频。平日都是关掉通知消息，今天颇有兴趣点开看。

哦，腰不错。大腿也不错。

退役以后，是不是都对运动短裤避之不及？

第二天一早，孙颖莎就有答案了，也不是。

电梯里只有他们俩个人。他穿着队服短裤，背着大大的包。

孙颖莎思忖，这人瘦了还是没瘦？肚子肉是没有了，大腿小腿饱满的肌肉线条还是半点不减昨天的视频风采。

爱情的力量。她咂舌。爱情的力量才会让人瘦下来爱上歌剧的魅力。

“孙颖莎。”他声音平静。

孙颖莎背后发冷。莫名听出了警告的意味。赶紧把赤裸直白的目光收回来，笑着伸出五只开花的手指：“是我是我！早上好！吃了吗？昨晚睡得好吗？”

她今天穿着圆领的白色短袖，黑色的超短裤下面是帆布鞋。看着像个学生。樊振东知道她不是。

“退役也没几年，记性倒是不行了。”

孙颖莎微笑。

“我退役的时候，也不知道你结婚了嘛！”她双手合十：“谁知当日一别，再见会是这番光景！你大人有大量！份子钱我给你补上！”

看她挤眉弄眼，说起话来机关枪一样。樊振东恍惚觉得她现在还应该跟以前一样，叽叽喳喳和别人一起随队训练。

回忆是软和的，嘴是硬邦的。

“不稀罕。”

孙颖莎撅嘴。在他后面下电梯。人家都不稀罕，她也不方便贴着去。毕竟当年她结婚，他也没来嘛。算起来，应该是她先结婚他在结婚的。

何况，她一退役，这人一番拉黑删除的操作，早就将俩人本就脆弱的队友情，一下斩断得干干净净了。

罢……

孙颖莎撞在他背上。绷紧的后背肌肉让她龇牙咧嘴。不是，这就是世界第一的背部肌肉群吗？

“你急刹车干嘛？”

在她怒目圆瞪的注视下，樊振东突然伸手帮她揉揉脑袋。孙颖莎看见他无名指上的戒指环，猛地缩着脑袋躲开。

“孙颖莎，你退役多久了？”

他表情认真得有些执拗。孙颖莎浅浅品尝到一丝受伤，在他那双波澜不惊的眼眸里。不过其实，他脸上没有表情。

孙颖莎晃晃脑袋，她最近得少看点言情小说了。

“三四年？”她嬉皮笑脸。

樊振东表情更冷了。不再说话抬步就走。

这不纯纯有病吗？跟当初删了她还拉黑的小学生行为有什么区别？孙颖莎扶额，小手揉揉自己脑袋。话说他这么结实，他娇娇柔柔的太太……

孙颖莎站住倒吸一口凉气。

傅清风说得没错，她就是个小色迷。

樊振东站在门口抽了一支烟，踩灭了捡起烟头才进去。运动场里乒乒乓乓击球，看见他问教练好。

想起她今天白色的帆布鞋，樊振东脑袋里全是白嫩脚趾上黑色的指甲油。她本身就是矛盾的结合体。

也是，欲望。

艹。

樊振东皱着眉头划拉手机，看见她和丁宁的合照发在丁宁朋友圈。她抱着小宝宝，被亲了个瓷实。小孩儿肉手贴在她挺翘的胸脯上。

樊振东把手机扔在包上，拿着球拍势必要把小队员训个七零八落。

前不久队里商务活动给他接了不少，现在这又归队忙起来，他都忘了本来他的生活跟孙颖莎，是没什么交集的。

孙颖莎想起那天樊振东第一个点赞，又取消。就有点好笑。

丁宁不解他一番鬼五十七的操作，孙颖莎可是知道他的别扭。不过她一向不会自讨没趣。

蓝旻看见孙颖莎的时候，打量的目光里多一点试探的意味。孙颖莎不想开口，等着她。

“莎莎，你们结婚多久了？”

孙颖莎回头看她，扎起来一个马尾，一下青春洋溢了起来。蓝旻应该是比自己小一点的，毕竟傅清风也比她小俩岁。

“三四年？”

她咬咬嘴唇。“我和东哥结婚的时候，还不知道你结婚了。”

孙颖莎看她纤瘦的身材，文弱的气质，怜爱之气由心而生。“这有什么关系，你们结婚我也不知道啊！”

她这话说得爽朗，笑声也清脆。电梯门一打开，樊振东听了个完全。脸色不变，却怎么都不肯看她一眼。

孙颖莎知道，得嘞，又抽抽。只是作为有妇之夫，能不能讲点武德，穿什么背心，搞什么肌肉诱惑？

樊振东接过蓝旻手里的袋子，不说话只往前面走。蓝旻有些试探的拉拉孙颖莎的手：“今晚来我家吃饭好吗？”

“我问问傅清风。”孙颖莎话倒是不假，傅清风工作忙，加班是常事儿。只是看蓝旻神色有些不太好了。

傅清风拒绝了。说是说有个会要开。孙颖莎坐在沙发上，看着看着综艺睡起了午觉。

朦朦胧胧醒起来肚子已经咕咕叫了。点了个外卖加急。拉开门去拿外卖，就看见樊振东弓着背在收拾袋子。

她提起外卖，刚好看见他侧头。

樊振东是听见外卖的声音出来的。门打开的时候，他就能猜到孙颖莎没穿鞋子。他看见俩只脚白白净净，在他的注视下，有些扭捏的扣紧了。黑色的趾甲冷漠又傲然，细细的丝绸吊带裙裙摆到大腿上，黑色，高贵又冷艳。

意外的称她。

他没敢往上看。光在黑色裙摆下的白皙大腿，就已经让他喉咙被扼住。

孙颖莎尴尬的不知道要不要打招呼，才想起自己没穿内衣。颇有些社死的冲他笑笑，看见他皱着眉转过了头。

一头的问号被关在门外。孙颖莎骂咧了一句，盘腿坐在地毯上吃外卖。

再弓下一点点，她的全部都会一览无余。

樊振东仰头，花洒冰凉的水扑在脸上。蓝旻站在浴室门外，低头不说话。

他下半身裹着浴巾拉开浴室的门，蓝旻抬起头，脸上一片泪痕。

“怎么了？”

“我害怕，我害怕。”她眼泪汪汪，仰着头看樊振东。可是第一次，他未能对她反复的情绪产生怜爱。

他给她擦干眼泪，指腹的触感却回忆着孙颖莎光洁的额头。“旻旻，跟我说怎么了，好不好？”

她摇头，只哭。

樊振东想抱住她，却想起自己上半身全是水。他们沉默地一个哭，一个擦眼泪。房间里的落地摆钟一声又一声。

（三）

孙颖莎最近忙起来了。忙着搞汇报，做高校讲座以及四处学习。傅清风都说她要赚钱养家了。孙颖莎想起那天他白色衬衣上的唇印，思忖了一下还是没问。

伸着腿，傅清风会意帮她按摩。

“我上个星期去出差，你一个电话都没给我打。”傅清风有些委屈地凑脸给她讨亲。

孙颖莎推开他的脸，哼哼俩声：“弟弟，我忙得很。”

他嘴角起飞，自己凑过去香她软绵绵的脸：“忙着安慰外面的弟弟吗？”

孙颖莎煞有介事的点头。傅清风往她身上拱。闹了一阵，孙颖莎趿着拖鞋去书房：“上周下雷雨不是下了好久吗，每天都是弟弟送我回家的。”

傅清风狗皮膏药一样黏上来。从背后抱住她腰，下巴搁在她肩膀上“孙颖莎，你欺负人。”

孙颖莎看着手上的戒指。下雨确实有人送。只是不是弟弟。

那天下大雨。素来外出不做任何准备的人，好不容易有兴致跟着学校里去游玩。下午日头还辣得很，她哪里知道带什么伞。

谁知道天刚黑打起雷来，闪电一道一道比白炽灯还亮。刚进小区没多走俩步，就淋了个兜头落汤鸡。

孙颖莎怕打雷害怕闪电。找了个没树的旮旯，抱着膝盖蹲着，走也不敢走，电话也不敢打。

樊振东的车停下。孙颖莎眼睛睁大了看他车灯亮着，大粒的雨里，他从驾驶座绕过车头，举着伞走到她面前。

黑色的大伞把他们遮住，孙颖莎湿漉漉的抬头看他。樊振东蹲下来，孙颖莎打了个颤。倒不是冷的。是被他眼神冻住的。

樊振东看她白色的宽松裙子下，完美的曲线暴露出来。贴在她身上，连成套的内衣都暴露出来。他把运动外套搭在她身上。

“呵，不做亏心事害怕什么雷。”樊振东看着她白白小小的手指光速从他手背上挪开。然后她干笑俩声。

“上车，我送你回去。”

“不用不用，我拿着伞走过去就行！”孙颖莎话刚说完，又闪过一道电。樊振东悠悠看着她，戏谑的表情一清二楚。

孙颖莎微笑。“那就谢谢你了。”他们并肩走着，樊振东和她胳膊贴着胳膊，她好像瘦了。

他撑伞把她送到副驾驶。孙颖莎嗖一下蹿进去。淋久了还是有点冷。樊振东收了伞，关上车门，把暖气打开。给她递了一块干净的毛巾。

孙颖莎舒了一口气，低头把裙摆的水沥干在毛巾里。搭在肩膀上的外套滑下来，她白裙下的姣好身材暴露无遗在光亮灯光的车厢里。“队里开会吗？穿这么正式。”

“你身材真好。”她笑眯眯。

无意的撩拨总是让人心痒难耐。她后背幽深的背脊沟将酒会的璀璨灯光盛摆着，摇曳生辉在与人自由交谈的从容里。

他们找一个代驾一起回家。十一月末的夜晚，陡然有些寒凉。她裹着长风衣，全然是樊振东不认识不了解的另一个模样。

风吹着她的头发，樊振东看着她的脸。醉得有些热。

他叫“孙颖莎。”

她站住。

他的手跟着风，去轻轻拨弄开她的头发，给她别在耳后。

“你为什么不跟我说一声？”

孙颖莎觉着他是喝醉了。他的眼睛里似乎带着点泪水，亮闪得孙颖莎失措：“说什么。”

她看着他的手扶着她的脸，他温柔地又有些遗憾地用指腹抚摸她的脸。

心跳一起失控。风把长青的树卷得沙沙响，落地未来得及扫的叶子在他们脚下卷起来。孙颖莎看着他裸露胸膛前松垮的领带，看他小手臂的肌肉，看他那双装满情绪的眼睛。

他低头笑：“我真蠢。”

孙颖莎想说点什么。他收回手，揣在裤兜里。“送你回家。”

蜿蜒的花园小路，一路吹着风踩着落叶。他陪她回家，同一部电梯，同一层楼。他们站在最远的斜对角，静默不语。

电梯一层一层升高，他叹气。孙颖莎套弄着手上的戒指，掩盖不合时宜的心跳如擂。

一进门，傅清风就抱了她一个满怀。孙颖莎笑眯眯踮着脚亲他耳垂：“怎么这么黏啊小狗狗～”

傅清风摸她发尾：“孙颖莎，一直和我在一起好不好啊？”

酒有点熏人，孙颖莎下巴搭在他肩膀上。微微打瞌睡。他听她呼吸绵软平缓，笑着把人抱起来，给她脱掉鞋子。洗了澡抱着人睡觉

手指在门框上扣紧。

他闭眼，就是她贴着他跳舞时候，灿烂说“你看我”的样子。他总忘不了她缠在傅清风腰上的腿还有小巧脚趾上的黑色指甲油。

蓝旻环住他的腰，贴在他宽厚的背上。“东哥，我明天在家等你吃饭好不好？”

樊振东把烟头灭在烟灰缸里。干涩的喉咙说好。

蓝旻感受眼泪滑过鼻梁，隐没在头发里。贪婪汲取他带着难闻烟味的气息，不真实的感觉像在迷雾里萎死的花朵。

（五）

孙颖莎发现傅清风把那件衬衣扔了。拍拍打呵欠大张着的嘴，她捡起来左看右看，就那一个唇印不至于把衣服扔了吧。

败家。

不过扔了就扔了。孙颖莎把衣服丢回去，又不是她的钱。吃一口面包就一口牛奶，她舒服得翘起脚趾。

真没意思这一天俩天的。乒协晚宴过后，十二月的天冷得仓促。她裹着棉袄从外面急急赶回来时，看见电梯里面色扭曲的樊振东。

他手扶着腰，神色痛苦。

孙颖莎咳嗽俩声，他也不理。她皱着眉，大大打了个喷嚏，他给了记冷刀。

不是，怎么有人不识好人心呢？

嗯。樊振东冷冷收回视线。

“孙颖莎。”

他警告。警告她不要逾矩，也警告自己，不要逾矩。

呵。孙颖莎用手捂住耳朵。

樊振东给她指洗手间。她洗完手，给他倒了杯水。樊振东看她坐在旁边喝水，有点好笑。

“孙颖莎，你满脑子黄色废料。”

樊振东坐起来，把衣服拉下去。

孙颖莎气愤的把杯子放下。“农夫与蛇！农夫与蛇！简直是农夫与蛇！”

她越过他要走，樊振东笑得无奈拉过她手腕，她一个没站稳，坐在他腿上。

真恶俗的情节。

艹。

艹。

（六）

天地良心。

基础知识

没错，使用博客也是需要有一些基础知识的。

  1. 两个config文件。在文件夹根目录下的config文件叫做站点配置文件，在themes/next下的config文件叫做主题配置文件，这两点需要分清。关于配置主题的更多内容，可以查看相关链接。

  2. 基本操作。在对内容进行修改过后，一定要在Blog文件夹-右键-服务-新建位于文件夹位置的终端窗口运行3个代码。

    1. hexo generate 这个代码是用来生成的。
    2. hexo deploy 这个代码是用来部署到Github服务器上的。
    3. hexo server 这个代码是用来生成预览界面的。

基本操作

接下来就是使用博客的基本操作。主要分为两部分：新建文章和插入图片。

新建文章

  1. 在Blog文件夹-右键-服务-新建位于文件夹位置的终端窗口运行代码hexo new post "新建文章的标题"。
  2. 在Blog/source/_posts里面查看新建的md。
  3. 一定要记住，在md的标题位置添加分类和标签，具体格式如下图所示。（tags即为标签，categories即为分类，格式如图所示）

插入图片

  1. 将图片拖入Blog/source/images
  2. 在Blog文件夹-右键-服务-新建位于文件夹位置的终端窗口运行hexo generate和hexo deploy两个代码。
  3. 在浏览器输入eefraa.github.io/images/图片名.jpg/.png/etc.，这个链接即是图片的链接。
  4. 在md当中输入！[图片描述]（图片的链接）即可插入图片。（注意：! 和 () 都要是半角的）
